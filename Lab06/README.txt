Book:

- author: char*
- title: string | char*
- pages_no: int
- page_words_count: int* - an integers array which stores the words count for each page of the book (page 0 is the summary page of the book)
- online: boolean (true if the book is in electronic format, false if the book is printed on paper only)

For this class implement:

- all the previously mentioned attributes: all attributes are PRIVATE
- constructor with parameters (one parameter for each class attribute)
- destructor
- copy constructor
- setters
- getters
- examples in MAIN function: create Book objects using the constructor with parameters, create objects using the copy constructor ...
